# ğŸ§  How the Stack Concept Works

This calculator follows the **stack principle used in expression evaluation**.

Basic workflow:

1. User inputs numbers and operators.
2. Expression is built dynamically.
3. When `=` is pressed:

   * The expression is evaluated.
4. Result is displayed and stored in history.

Stack-based calculators are widely used in:

* Compilers
* Expression parsing
* Mathematical interpreters
